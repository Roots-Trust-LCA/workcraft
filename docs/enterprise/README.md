## Value Proposition

**World of Workcraft provides organizational infrastructure that makes work legible:**

- Clear engagement types and scope classification
- Visible capability development and career progression
- Transparent economic flows and contribution tracking
- Structured cross-functional collaboration without hierarchy

**Outcomes:**
- Local autonomy with global coordination
- Economic alignment with contribution
- Resilient infrastructure (no central point of failure)
- Professional development that's portable and visible
- Emergent coordination (standards arise from practice)

## Professional Vocabulary

World of Workcraft uses organizational terminology, not gaming metaphors:

| Instead Of | Use |
|------------|-----|
| Quest | Engagement / Initiative |
| Raid | Cross-Functional Initiative |
| Guild | Practice Community |
| Class | Role / Function |
| XP | Contribution Credits |
| Level | Professional Stage |
| Loot | Allocation / Distribution |
| Mana Pool | Capacity Budget |

See [Vocabulary Reference](./VOCABULARY_REFERENCE.md) for complete translation guide.
